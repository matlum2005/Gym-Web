# Gym Web

A responsive and professional Gym Management Website built with HTML, CSS, JavaScript, Bootstrap, and Tailwind CSS.

## Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **AI-Powered Smart Tools**: BMI Calculator, AI Workout Recommendations, and Interactive Chatbot
- **Photo Gallery**: Showcase of gym facilities and equipment
- **Upcoming Events & Classes**: Schedule and booking for various fitness classes
- **Training Programs**:
  - Strength & Conditioning
  - High Intensity Training (HIIT)
  - Recovery & Mobility Sessions
- **Trainer Profiles**: Meet our expert coaches and book sessions
- **Member Testimonials**: Real feedback from our community
- **Membership Plans**: Flexible pricing options (Basic, Standard, Premium)
- **Contact & Location**: Integrated map and contact form
- **User Authentication**: Secure login and registration system
- **Admin Dashboard**: Management tools for administrators
- **Backend API**: RESTful services built with Spring Boot for data management
